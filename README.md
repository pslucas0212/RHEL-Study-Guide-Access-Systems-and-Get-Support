### Edit Text Files
Use vi or vim to edit text files
```
$ vim filename
```

Four modes of vim:
1. Edit mode - insert
2. Command mode
3. Visual edit mode
4. Extended command mode

Configuration file for vim is located in:
```
$ cat ~/.vimrc
```
